# Fuel Ledger

A step-by-step nutrition planner: enter your stats, pick a goal, compare a few paces, get a macro breakdown. All calculated live in the browser, vanilla JS only.

**Live site:** [Add your GitHub Pages link here once deployed]

## What it does

1. **Your profile** → calculates maintenance calories (Mifflin-St Jeor + activity multiplier + workout bonus)
2. **Your goal** → target weight, timeframe, cut/bulk
3. **Plan options** → 4 cards (your timeframe + Conservative/Moderate/Aggressive paces), with warnings on anything too aggressive
4. **Macro breakdown** → pick a plan + style (Lean/Lean Muscle/Bulk Muscle) → protein/carb/fat grams

Each step recalculates from the one before it, so changing an earlier step resets anything built on the old numbers.

## How to use it

Fill in Step 1 → calculate. Step 2 unlocks → build your plan. Step 3 unlocks → pick a card. Step 4 unlocks → pick a style. "Reset everything" in the header starts over.
